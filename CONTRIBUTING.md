# 贡献指南

感谢你对 AI Virtual Cell 项目的兴趣！我们欢迎各种形式的贡献。

## 🤝 如何贡献

### 1. 参与讨论

前往 [GitHub Discussions](https://github.com/AICode-Nexus/ai-virtual-cell/discussions) 参与讨论：

- **生物学验证**：帮助验证生物学机制的准确性
- **技术实现**：探讨技术可行性和实现细节
- **应用场景**：探索实际应用和商业价值
- **创新想法**：分享你的想法和建议

### 2. 提交 Issue

发现问题或有建议？[创建 Issue](https://github.com/AICode-Nexus/ai-virtual-cell/issues/new)

请包含：
- 清晰的标题
- 详细的描述
- 相关的上下文信息

### 3. 改进文档

文档改进步骤：
1. Fork 本仓库
2. 创建分支：`git checkout -b improve-docs`
3. 修改文档
4. 提交 PR

### 4. 代码贡献（未来）

当项目进入实现阶段，我们会欢迎代码贡献。

## 📋 贡献者公约

我们承诺为所有人提供友好、安全和包容的环境。请遵守以下原则：

- 尊重不同观点和经验
- 接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

## 🎯 我们特别需要

- 🧬 **生物学家**：验证细胞机制映射
- 💻 **AI 研究者**：评估技术可行性
- 🏗️ **架构师**：优化系统设计
- 🎨 **产品经理**：探索应用场景
- 📝 **技术写作者**：改进文档质量

## 📧 联系方式

- GitHub Discussions: [讨论区](https://github.com/AICode-Nexus/ai-virtual-cell/discussions)
- Issues: [问题追踪](https://github.com/AICode-Nexus/ai-virtual-cell/issues)

感谢你的贡献！🙏
