# 免疫系统 (Immune System)

> 先天免疫、适应性免疫、免疫记忆与耐受、自我/非我识别、免疫突触与细胞间协作、免疫逃逸与肿瘤免疫

## 1. 系统概述

免疫系统是多细胞生物体识别和清除病原体、异常细胞与损伤组织的核心防御层，负责：
- **快速响应**：通过先天免疫在数小时内识别病原体模式并启动炎症反应
- **精确识别**：通过适应性免疫生成高特异性受体，识别特定抗原并清除感染细胞
- **长期记忆**：通过记忆细胞在再次感染时快速启动强效应答
- **自我耐受**：通过中枢和外周耐受机制避免攻击自身组织
- **免疫监视**：通过持续巡逻识别并清除癌变细胞和衰老细胞
- **组织修复**：通过调节性免疫细胞和抗炎因子促进损伤后的组织重建

该系统不是单一防御线，而是由先天免疫的快速模式识别、适应性免疫的精确克隆扩增、记忆细胞的长期保护和耐受机制的自我约束共同构成的多层防御网络。

## 2. 核心机制详解

### 2.1 先天免疫（Innate Immunity）

#### 2.1.1 模式识别受体（PRR）

- **Toll样受体（TLR）识别病原体相关分子模式（PAMP）**：TLR4识别脂多糖（LPS），TLR3识别双链RNA，TLR9识别未甲基化CpG DNA，触发NF-κB和IRF通路激活炎症基因。
- **NOD样受体（NLR）监测胞内病原体**：NOD1/2识别肽聚糖片段，NLRP3感知ATP、尿酸结晶、病原体毒素，组装炎症小体激活caspase-1，切割pro-IL-1β和pro-IL-18为活性形式。
- **RIG-I样受体（RLR）检测病毒RNA**：RIG-I和MDA5识别胞质中的病毒RNA，通过MAVS激活IRF3/7，诱导I型干扰素（IFN-α/β）表达，建立抗病毒状态。
- **C型凝集素受体（CLR）识别真菌和细菌多糖**：Dectin-1识别β-葡聚糖，触发吞噬作用和炎症因子释放，介导抗真菌免疫。

#### 2.1.2 补体系统

- **经典途径由抗体-抗原复合物激活**：C1q结合IgG或IgM，启动C1r-C1s级联，切割C4和C2，形成C3转化酶（C4b2a）。
- **旁路途径由病原体表面直接激活**：C3自发水解后结合因子B和D，形成C3转化酶（C3bBb），在病原体表面放大C3b沉积。
- **凝集素途径由甘露糖结合凝集素（MBL）启动**：MBL识别病原体表面甘露糖，激活MASP-1/2，切割C4和C2，汇入经典途径。
- **C3b调理作用促进吞噬**：C3b沉积在病原体表面，被吞噬细胞的补体受体（CR1/CR3）识别，增强吞噬效率。
- **膜攻击复合体（MAC）直接杀伤**：C5b-C9组装形成跨膜孔道，导致细胞膜破裂和病原体裂解。
- **过敏毒素C3a和C5a招募炎症细胞**：结合G蛋白偶联受体，诱导血管通透性增加、肥大细胞脱颗粒和中性粒细胞趋化。

#### 2.1.3 吞噬作用与抗原呈递

- **吞噬细胞通过PRR和调理素受体识别病原体**：巨噬细胞和中性粒细胞表达TLR、清道夫受体、Fc受体和补体受体，多途径识别并内吞病原体。
- **吞噬体-溶酶体融合形成吞噬溶酶体**：内吞后吞噬体与溶酶体融合，释放水解酶、活性氧（ROS）和活性氮（RNS），降解病原体。
- **NADPH氧化酶产生呼吸爆发**：吞噬细胞激活后NADPH氧化酶（NOX2）将氧转化为超氧阴离子，进一步生成H2O2和次氯酸，杀伤病原体。
- **抗原加工与MHC II类呈递**：吞噬溶酶体中的蛋白被降解为肽段，装载到MHC II类分子，转运至细胞表面呈递给CD4+ T细胞。

#### 2.1.4 自然杀伤细胞（NK细胞）

- **NK细胞通过"缺失自我"识别靶细胞**：正常细胞表达MHC I类分子，通过抑制性受体（KIR、NKG2A）抑制NK细胞；病毒感染或肿瘤细胞下调MHC I，解除抑制，激活NK细胞杀伤。
- **激活性受体识别应激配体**：NKG2D识别MICA/MICB（应激诱导配体），NCR识别病毒血凝素，激活NK细胞释放穿孔素和颗粒酶。
- **抗体依赖的细胞介导的细胞毒性（ADCC）**：NK细胞通过Fc受体（CD16）识别抗体包被的靶细胞，释放细胞毒性颗粒杀伤靶细胞。
- **NK细胞分泌IFN-γ调节免疫应答**：激活后NK细胞快速产生IFN-γ，激活巨噬细胞，促进Th1分化，增强细胞免疫。

### 2.2 适应性免疫（Adaptive Immunity）

#### 2.2.1 T细胞激活

- **T细胞受体（TCR）识别MHC-肽复合体**：CD8+ T细胞识别MHC I类-肽（内源抗原），CD4+ T细胞识别MHC II类-肽（外源抗原），提供信号1。
- **共刺激信号决定激活或无能**：CD28结合B7（CD80/CD86）提供信号2，与信号1协同激活T细胞；缺乏信号2导致T细胞无能（anergy）。
- **细胞因子提供信号3塑造分化方向**：IL-12促进Th1分化（IFN-γ产生），IL-4促进Th2分化（IL-4/IL-5/IL-13产生），TGF-β+IL-6促进Th17分化（IL-17产生）。
- **T细胞克隆扩增**：激活后T细胞进入快速增殖，单个T细胞可在7-10天内扩增至10^4-10^5个效应细胞。

#### 2.2.2 B细胞激活与抗体生成

- **B细胞受体（BCR）识别天然抗原**：膜结合IgM/IgD识别可溶性或膜结合抗原，内吞并加工为肽段，通过MHC II类呈递给辅助性T细胞（Tfh）。
- **T依赖性B细胞激活需要T-B协作**：Tfh通过CD40L-CD40相互作用和细胞因子（IL-21）激活B细胞，诱导生发中心形成。
- **生发中心反应产生高亲和力抗体**：B细胞在生发中心经历体细胞超突变（SHM），亲和力成熟选择高亲和力克隆，类别转换重组（CSR）改变抗体类型（IgM→IgG/IgA/IgE）。
- **浆细胞分泌大量抗体**：分化的浆细胞每秒可分泌数千个抗体分子，维持数天至数月，提供体液免疫保护。

#### 2.2.3 克隆选择理论

- **每个淋巴细胞表达单一特异性受体**：通过V(D)J重组，每个T/B细胞生成独特的TCR/BCR，理论多样性达10^15-10^18。
- **抗原选择性扩增特异性克隆**：抗原刺激后，识别该抗原的淋巴细胞克隆扩增，其他克隆保持静息，实现特异性免疫应答。
- **效应细胞清除抗原后大部分凋亡**：应答高峰后95%以上效应细胞通过凋亡清除，避免过度炎症和自身免疫。
- **记忆细胞长期存活提供保护**：5-10%效应细胞分化为记忆细胞，维持数年至终生，再次感染时快速扩增。

### 2.3 免疫记忆与耐受（Immune Memory and Tolerance）

#### 2.3.1 免疫记忆

- **记忆T细胞分为中枢记忆（TCM）和效应记忆（TEM）**：TCM表达CCR7和CD62L，归巢至淋巴器官，增殖能力强；TEM归巢至外周组织，快速产生效应分子。
- **组织驻留记忆T细胞（TRM）提供局部保护**：TRM表达CD69和CD103，长期驻留在皮肤、肺、肠道等屏障组织，再次感染时快速响应。
- **记忆B细胞快速分化为浆细胞**：再次抗原刺激后，记忆B细胞在2-3天内分化为浆细胞，比初次应答快5-7天。
- **长寿命浆细胞维持基础抗体水平**：驻留在骨髓的长寿命浆细胞持续分泌抗体，维持数年至终生的体液免疫保护。

#### 2.3.2 中枢耐受

- **胸腺阴性选择清除自身反应性T细胞**：胸腺髓质上皮细胞（mTEC）通过AIRE表达组织特异性抗原，高亲和力识别自身抗原的T细胞被诱导凋亡。
- **胸腺阳性选择保留识别自身MHC的T细胞**：皮质上皮细胞呈递自身肽-MHC复合体，中等亲和力识别的T细胞存活，完全不识别的T细胞因缺乏生存信号而死亡。
- **骨髓B细胞受体编辑和克隆清除**：自身反应性B细胞通过受体编辑改变BCR特异性，编辑失败则被诱导凋亡或进入无能状态。

#### 2.3.3 外周耐受

- **调节性T细胞（Treg）抑制自身免疫**：Foxp3+ Treg通过IL-10、TGF-β和CTLA-4抑制效应T细胞，维持免疫稳态。
- **无共刺激信号诱导T细胞无能**：外周组织中抗原呈递细胞（APC）缺乏共刺激分子，T细胞接受TCR信号但无CD28信号，进入无能状态。
- **免疫检查点抑制过度激活**：CTLA-4竞争性结合B7，阻断CD28信号；PD-1结合PD-L1/L2，抑制T细胞激活和增殖，防止组织损伤。

### 2.4 自我/非我识别（Self/Non-Self Recognition）

#### 2.4.1 MHC呈递系统

- **MHC I类分子呈递内源抗原**：胞质蛋白被蛋白酶体降解，肽段通过TAP转运至内质网，装载到MHC I类分子，呈递给CD8+ T细胞。
- **MHC II类分子呈递外源抗原**：内吞的蛋白在内体/溶酶体降解，肽段装载到MHC II类分子，呈递给CD4+ T细胞。
- **交叉呈递连接先天和适应性免疫**：树突状细胞（DC）可将外源抗原装载到MHC I类分子，激活CD8+ T细胞，对抗病毒感染和肿瘤。
- **MHC多态性增强群体免疫**：人类MHC（HLA）基因高度多态，不同个体呈递不同肽段谱，增加群体对病原体的抵抗力。

#### 2.4.2 TCR和BCR重排

- **V(D)J重组生成受体多样性**：RAG1/RAG2重组酶介导V、D、J基因片段随机重组，TdT添加非模板核苷酸，生成10^15-10^18种不同受体。
- **等位排斥确保单一特异性**：成功重排一条染色体后，反馈抑制另一条染色体重排，确保每个细胞只表达一种TCR/BCR。
- **生产性重排检查点**：只有正确读框的重排产生功能性受体，非生产性重排的细胞被清除。

### 2.5 免疫突触与细胞间协作（Immunological Synapse and Cell-Cell Cooperation）

#### 2.5.1 APC-T细胞免疫突触

- **超分子激活簇（SMAC）结构**：中央SMAC（cSMAC）富集TCR-MHC复合体和CD28-B7，外周SMAC（pSMAC）富集整合素LFA-1-ICAM-1，稳定细胞接触。
- **信号放大与持续激活**：免疫突触聚集TCR和共刺激分子，降低激活阈值，延长信号持续时间，确保完全激活。
- **极化分泌定向杀伤**：细胞毒性T细胞（CTL）形成免疫突触后，将穿孔素和颗粒酶定向释放至靶细胞，避免旁观者杀伤。

#### 2.5.2 T-B细胞协作

- **Tfh细胞归巢至生发中心**：表达CXCR5的Tfh细胞响应CXCL13梯度，迁移至B细胞滤泡，提供帮助信号。
- **CD40L-CD40相互作用是关键信号**：Tfh表达CD40L，结合B细胞CD40，激活NF-κB通路，诱导B细胞增殖、类别转换和生发中心形成。
- **IL-21促进B细胞分化**：Tfh分泌IL-21，促进B细胞分化为浆细胞和记忆B细胞，增强抗体应答。

#### 2.5.3 细胞因子网络

- **Th1-Th2交叉抑制**：IFN-γ抑制Th2分化，IL-4抑制Th1分化，形成互斥的免疫应答模式。
- **细胞因子级联放大信号**：IL-12诱导NK和T细胞产生IFN-γ，IFN-γ激活巨噬细胞产生更多IL-12，形成正反馈。
- **抗炎细胞因子终止应答**：IL-10和TGF-β抑制促炎细胞因子产生，促进Treg分化，防止过度炎症。

### 2.6 免疫逃逸与肿瘤免疫（Immune Evasion and Tumor Immunity）

#### 2.6.1 免疫编辑三阶段

- **清除阶段**：免疫系统识别并清除大部分肿瘤细胞，NK细胞和CTL发挥主要作用。
- **平衡阶段**：残留肿瘤细胞与免疫系统达到动态平衡，免疫压力选择低免疫原性变异株。
- **逃逸阶段**：肿瘤细胞积累免疫逃逸机制，突破免疫监视，形成临床可检测肿瘤。

#### 2.6.2 免疫检查点抑制

- **肿瘤细胞上调PD-L1**：IFN-γ诱导肿瘤细胞表达PD-L1，结合T细胞PD-1，抑制T细胞激活和杀伤功能。
- **CTLA-4阻断T细胞启动**：肿瘤微环境中Treg高表达CTLA-4，竞争性结合B7，阻断CD28共刺激信号。
- **免疫检查点阻断疗法**：抗PD-1/PD-L1和抗CTLA-4抗体解除T细胞抑制，恢复抗肿瘤免疫，在多种癌症中显示疗效。

#### 2.6.3 肿瘤微环境免疫抑制

- **肿瘤相关巨噬细胞（TAM）促进肿瘤生长**：M2型TAM分泌VEGF、TGF-β和IL-10，促进血管生成、抑制T细胞和重塑基质。
- **髓系来源抑制细胞（MDSC）抑制T细胞**：MDSC通过精氨酸酶、iNOS和ROS抑制T细胞增殖和功能。
- **调节性T细胞富集**：肿瘤分泌TGF-β和CCL22，招募Treg至肿瘤微环境，抑制效应T细胞。
- **代谢竞争限制T细胞功能**：肿瘤细胞高糖酵解消耗葡萄糖，产生乳酸酸化微环境，抑制T细胞代谢和功能。

## 3. 生物学意义

- **多层防御确保生存**：先天免疫提供快速但非特异性保护，适应性免疫提供精确但延迟的清除，记忆细胞提供长期保护。
- **自我耐受防止自身免疫**：中枢和外周耐受机制清除或抑制自身反应性淋巴细胞，维持免疫稳态。
- **免疫记忆是疫苗的基础**：记忆细胞的长期存活和快速召回使疫苗能够预防感染性疾病。
- **免疫监视清除癌变细胞**：持续巡逻的免疫细胞识别并清除早期癌变细胞，降低肿瘤发生率。
- **免疫失调是多种疾病的核心**：过度激活导致自身免疫病和过敏，功能不足导致感染易感和肿瘤，失衡是炎症性疾病的根源。

## 4. 调控输入、输出与约束

### 4.1 输入
- 病原体相关分子模式（PAMP）：LPS、肽聚糖、病毒核酸
- 损伤相关分子模式（DAMP）：ATP、HMGB1、尿酸结晶
- 抗原：蛋白质、多糖、脂质、核酸
- 细胞因子和趋化因子信号
- 共刺激和共抑制信号

### 4.2 输出
- 炎症反应（细胞因子、趋化因子、急性期蛋白）
- 病原体清除（吞噬、细胞毒性、抗体中和）
- 免疫记忆建立（记忆T/B细胞）
- 组织修复和重塑
- 免疫耐受维持

### 4.3 调控约束
- **抗原剂量约束**：低剂量抗原诱导耐受，高剂量诱导免疫应答。
- **共刺激依赖约束**：缺乏共刺激信号导致T细胞无能，防止不适当激活。
- **检查点抑制约束**：CTLA-4和PD-1限制T细胞过度激活，防止自身免疫和组织损伤。
- **代谢约束**：淋巴细胞激活需要糖酵解和氧化磷酸化，代谢底物限制免疫应答强度。
- **空间约束**：淋巴器官提供结构化微环境，促进APC-T-B细胞相互作用，外周组织缺乏此结构限制局部免疫应答。

## 5. 失败模式与病理后果

- **先天免疫缺陷**：TLR、补体或吞噬细胞缺陷导致反复细菌和真菌感染。
- **适应性免疫缺陷**：SCID（严重联合免疫缺陷）、X-连锁无丙种球蛋白血症导致严重感染易感。
- **自身免疫病**：中枢或外周耐受失败导致类风湿关节炎、系统性红斑狼疮、1型糖尿病。
- **过敏反应**：Th2偏倚和IgE介导的肥大细胞脱颗粒导致过敏性鼻炎、哮喘、过敏性休克。
- **慢性炎症**：持续抗原刺激或调节失衡导致炎症性肠病、动脉粥样硬化、神经退行性疾病。
- **免疫衰老**：胸腺萎缩、记忆细胞积累、炎症因子升高导致老年人感染易感和疫苗应答减弱。
- **移植排斥**：同种异体MHC差异触发T细胞介导的急性或慢性排斥反应。
- **肿瘤免疫逃逸**：MHC下调、PD-L1上调、免疫抑制微环境形成导致肿瘤逃避免疫监视。
- **细胞因子风暴**：过度免疫激活（如严重感染、CAR-T治疗）导致IL-6、TNF-α大量释放，引发多器官衰竭。
- **免疫检查点抑制剂相关不良反应**：解除免疫抑制导致自身免疫样毒性（结肠炎、肺炎、甲状腺炎）。
