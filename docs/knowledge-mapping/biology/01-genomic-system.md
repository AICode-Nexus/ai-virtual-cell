# 基因组系统 (Genomic System)

> DNA复制、修复、重组、染色质重塑与表观遗传调控

## 1. 系统概述

基因组系统是细胞信息存储和传递的核心，负责：
- **信息存储**：DNA作为遗传信息的载体
- **信息复制**：确保遗传信息准确传递给子代
- **信息保护**：修复DNA损伤，维持基因组稳定性
- **信息调控**：通过染色质结构和表观遗传修饰调控基因表达

## 2. 核心机制详解

### 2.1 DNA复制机制

#### 2.1.1 复制起始
- **复制起点（Origin of Replication）**
  - 原核生物：单一起点（oriC）
  - 真核生物：多个起点（每10-100kb一个）
  - 起点识别复合物（ORC）识别并结合起点
  
- **解旋酶复合物**
  - MCM2-7复合物：DNA解旋酶
  - 解开双螺旋，形成复制叉
  - ATP驱动的马达蛋白

#### 2.1.2 复制延伸
- **DNA聚合酶家族**
  - DNA Pol α：合成RNA引物和短DNA片段
  - DNA Pol δ：合成滞后链（冈崎片段）
  - DNA Pol ε：合成前导链
  - 校对功能：3'→5'外切酶活性

- **前导链与滞后链**
  - 前导链：连续合成（5'→3'方向）
  - 滞后链：不连续合成（冈崎片段，100-200nt）
  - 引物酶：合成RNA引物
  - DNA连接酶：连接冈崎片段

#### 2.1.3 复制保真性
- **校对机制**
  - DNA聚合酶的3'→5'外切酶活性
  - 错配率：10⁻⁷ per base
  
- **错配修复（MMR）**
  - MutS识别错配
  - MutL招募修复酶
  - 切除并重新合成
  - 最终错配率：10⁻¹⁰ per base

### 2.2 DNA修复机制

#### 2.2.1 碱基切除修复（BER）
- **适用场景**：单个碱基损伤（氧化、脱氨基）
- **机制**：
  1. DNA糖基化酶识别并切除损伤碱基
  2. AP核酸内切酶切开磷酸二酯键
  3. DNA聚合酶填补缺口
  4. DNA连接酶封闭切口

#### 2.2.2 核苷酸切除修复（NER）
- **适用场景**：大块损伤（UV诱导的嘧啶二聚体）
- **机制**：
  1. 损伤识别（XPC-RAD23B复合物）
  2. 解旋（TFIIH复合物）
  3. 切除损伤片段（24-32nt）
  4. 填补合成
  5. 连接

#### 2.2.3 双链断裂修复（DSB Repair）
- **同源重组修复（HR）**
  - 高保真，需要姐妹染色单体作为模板
  - 关键蛋白：RAD51、BRCA1/2
  - 主要在S/G2期

- **非同源末端连接（NHEJ）**
  - 快速但易出错
  - 关键蛋白：Ku70/80、DNA-PKcs、Ligase IV
  - 全细胞周期可用

#### 2.2.4 错配修复（MMR）
- **识别错配**：MutS同源蛋白（MSH2-MSH6）
- **招募修复**：MutL同源蛋白（MLH1-PMS2）
- **切除重合成**：EXO1外切酶、DNA聚合酶

### 2.3 DNA重组机制

#### 2.3.1 同源重组（Homologous Recombination）
- **减数分裂重组**
  - 交叉互换（Crossover）
  - 基因转换（Gene Conversion）
  - SPO11诱导双链断裂
  - 增加遗传多样性

- **体细胞重组**
  - V(D)J重组（免疫系统）
  - 类别转换重组（抗体类型切换）
  - RAG1/RAG2重组酶

#### 2.3.2 位点特异性重组
- **整合酶介导**：λ噬菌体整合
- **转座子**：DNA片段的移动
- **CRISPR系统**：细菌的适应性免疫

### 2.4 染色质重塑

#### 2.4.1 核小体结构
- **组成**：
  - 核心：8个组蛋白（H2A、H2B、H3、H4各2个）
  - DNA：147bp缠绕核心1.65圈
  - 连接组蛋白H1：稳定核小体间DNA

- **动态性**：
  - 核小体滑动
  - 核小体弹出/重组装
  - 组蛋白变体替换

#### 2.4.2 染色质重塑复合物
- **SWI/SNF家族**
  - ATP依赖的染色质重塑
  - 滑动、弹出核小体
  - 激活基因转录

- **ISWI家族**
  - 核小体间距调节
  - 染色质组装

- **CHD家族**
  - 染色质解凝聚
  - 转录调控

- **INO80家族**
  - 组蛋白变体交换
  - DNA修复

#### 2.4.3 染色质高级结构
- **30nm纤维**：核小体串珠的螺旋化
- **染色质环**：CTCF和cohesin介导
- **拓扑关联域（TAD）**：基因组的功能单元
- **染色体区室**：A区室（活跃）vs B区室（沉默）

### 2.5 表观遗传调控

#### 2.5.1 DNA甲基化
- **机制**：
  - DNA甲基转移酶（DNMT1、DNMT3A/B）
  - 5-甲基胞嘧啶（5mC）
  - 主要在CpG二核苷酸

- **功能**：
  - 基因沉默
  - X染色体失活
  - 基因组印记
  - 转座子抑制

- **去甲基化**：
  - TET酶：5mC → 5hmC → 5fC → 5caC
  - BER途径移除修饰碱基

#### 2.5.2 组蛋白修饰
- **乙酰化（Acetylation）**
  - 组蛋白乙酰转移酶（HAT）
  - 去乙酰化酶（HDAC）
  - 中和正电荷，松弛染色质
  - 激活转录

- **甲基化（Methylation）**
  - H3K4me3：启动子活跃标记
  - H3K27me3：Polycomb抑制标记
  - H3K9me3：异染色质标记
  - H3K36me3：转录延伸标记

- **磷酸化（Phosphorylation）**
  - H3S10ph：染色体凝聚
  - H2AXS139ph（γH2AX）：DNA损伤响应

- **泛素化（Ubiquitination）**
  - H2AK119ub：Polycomb抑制
  - H2BK120ub：转录延伸

#### 2.5.3 非编码RNA调控
- **长非编码RNA（lncRNA）**
  - XIST：X染色体失活
  - HOTAIR：Polycomb招募
  - 染色质结构调控

- **小RNA**
  - miRNA：转录后调控
  - siRNA：基因沉默
  - piRNA：转座子抑制

### 2.6 基因组稳定性维护

#### 2.6.1 端粒维护
- **端粒结构**：
  - TTAGGG重复序列（人类）
  - T-loop结构保护染色体末端
  - Shelterin复合物

- **端粒酶**：
  - 逆转录酶（TERT）
  - RNA模板（TERC）
  - 延长端粒，对抗复制末端问题

#### 2.6.2 着丝粒功能
- **结构**：
  - α-卫星DNA重复
  - CENP-A组蛋白变体
  - 动粒蛋白复合物

- **功能**：
  - 姐妹染色单体连接
  - 纺锤体附着
  - 染色体分离检查点

#### 2.6.3 基因组监控
- **DNA损伤检查点**
  - ATM/ATR激酶
  - p53肿瘤抑制因子
  - 细胞周期阻滞或凋亡

- **复制应激响应**
  - 复制叉停滞检测
  - 复制叉保护和重启
  - 防止基因组不稳定

## 3. 生物学意义

### 3.1 遗传信息的忠实传递
- 确保子代细胞获得完整准确的遗传信息
- 维持物种的遗传稳定性

### 3.2 适应性与进化
- 突变提供遗传变异
- 重组增加遗传多样性
- 自然选择作用的基础

### 3.3 基因表达调控
- 表观遗传修饰实现基因选择性表达
- 细胞分化和发育的分子基础
- 环境响应的记忆机制

### 3.4 疾病相关
- DNA修复缺陷 → 癌症易感性
- 表观遗传异常 → 多种疾病
- 基因组不稳定 → 衰老和退行性疾病
